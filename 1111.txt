========================================
数据修改请求流程 - 问答文档
========================================

问题1：当修改一条数据时，点击确认，发送了什么请求？
回答：POST 请求

问题2：发送的数据是什么？
回答：JSON 格式的文章数据，包含 id, title, author, pageviews, status, content, display_time 等字段
代码位置：src/api/article.js 第15-16行
export function updateArticle(id, data) {
  return request({ url: '/vue-admin-template/table/update', method: 'post', data: { id, ...data } })
}

问题3：请求的 URL 是什么？
回答：/vue-admin-template/table/update

问题4：请求发给了哪个服务器？前端还是后端？
回答：先发给前端开发服务器（http://localhost:9528），再通过代理转发到后端 Flask 服务器（http://localhost:5000）

问题5：有没有进行转发？
回答：有转发

问题6：转发规则是什么？
回答：所有以 /vue-admin-template 开头的请求转发到 http://localhost:5000
代码位置：vue.config.js 第39-43行
proxy: {
  '/vue-admin-template': {
    target: 'http://localhost:5000',
    changeOrigin: true
  }
}

问题7：转发规则在哪个文件设置？
回答：vue.config.js 文件中的 devServer.proxy 配置

问题8：后端哪个视图函数处理了该请求？
回答：app.py 第237行的 update_article 函数
@app.route('/vue-admin-template/table/update', methods=['POST'])

问题9：数据库的表是不是基于 SQLAlchemy 实现的？
回答：不是。使用的是 PyMySQL 原生 SQL 查询
代码位置：models.py 第47-60行
def get_db_connection():
    connection = pymysql.connect(
        host=Config.MYSQL_HOST,
        port=Config.MYSQL_PORT,
        user=Config.MYSQL_USER,
        password=Config.MYSQL_PASSWORD,
        database=Config.MYSQL_DB,
        charset='utf8mb4',
        cursorclass=pymysql.cursors.DictCursor
    )
    return connection

问题10：如果查询语句包含原始 SQL，如何改成基于 ORM 的方式？
回答：改用 SQLAlchemy ORM 方式示例：
from sqlalchemy import create_engine, Column, Integer, String
from sqlalchemy.orm import declarative_base, sessionmaker

Base = declarative_base()

class Article(Base):
    __tablename__ = 'articles'
    id = Column(Integer, primary_key=True)
    title = Column(String(255))
    author = Column(String(50))

engine = create_engine('mysql+pymysql://user:pass@localhost/db')
Session = sessionmaker(bind=engine)

def update_article_orm(article_id, **kwargs):
    session = Session()
    article = session.query(Article).filter_by(id=article_id).first()
    if article:
        for key, value in kwargs.items():
            setattr(article, key, value)
        session.commit()
    session.close()

问题11：视图函数返回了什么格式的数据给前端？
回答：JSON 格式
代码位置：app.py 第295-303行
成功：return jsonify({'code': 20000, 'data': 'success'})
失败：return jsonify({'code': 50005, 'message': '更新操作未生效'}), 500

问题12：前端哪段代码用来接收后端发来的数据？
回答：通过 Axios 响应拦截器接收
代码位置：src/utils/request.js 第33-50行
service.interceptors.response.use(
  response => {
    const res = response.data  // 接收后端 JSON 数据
    if (res.code !== 20000) {
      Message({ message: res.message || 'Error', type: 'error' })
    }
    return res
  }
)
